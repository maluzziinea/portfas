MINHAS HABILIDADES:

CSS é a sigla para o termo em inglês Cascading Style Sheets que, traduzido para o português, significa Folha de Estilo em Cascatas. O CSS é fácil de aprender e entender e é facilmente utilizado com as linguagens de marcação HTML ou XHTML.

A Linguagem de Marcação de Hipertexto (HTML) é uma linguagem de computador que compõe a maior parte das páginas da internet e dos aplicativos online.

O JavaScript surgiu como uma tecnologia do lado do navegador para tornar as aplicações Web mais dinâmicas. Ao usar JavaScript, os navegadores passaram a ser capazes de responder a interações do usuário e alterar o layout do conteúdo na página.

Node, ou NodeJS, é o Back-End do JavaScript, ou seja, a parte de trás da linguagem. Com o Node podemos utilizar e crias APIs, por exemplo.

GIT é um sistema de controle de histórico de arquivos aberto onde todos têm acesso ao repositório para que caso alguma alteração precise ser feita, ou até mesmo desfeita, todos podem acessá-lo.

GitHub é uma plataforma onde podemos gerenciar códigos, repositórios e compartilhá-los com outros desenvolvedores.
